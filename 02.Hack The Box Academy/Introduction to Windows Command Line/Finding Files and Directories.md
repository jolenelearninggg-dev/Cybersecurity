## Finding Files and Directories

`where /R [路徑] [檔名]`：遞迴搜尋特定檔案，能找到檔案路徑。 
- ➔ 電腦的 「追蹤器」

`findstr`：進階內容字串搜尋，支援正規表示法 (Regex)，可以搜尋符合指定的規則的內容。 若只是用`find`也可單純搜尋檔案內容。
- ➔ 電腦的 「高級篩選器」（類似 Linux 的 grep）

`fc /N [檔案1] [檔案2]`：比對檔案並顯示完整有差異的內容與行號。 若只是用`comp`只顯示錯誤的位置。 
- ➔ 電腦的 「差異分析儀」（比 comp 更直觀）

`sort /unique`：將內容排序並刪除重複項。 使用`sort`可單純排序，不刪除重複項。
- ➔ 電腦的 「自動收納盒」
