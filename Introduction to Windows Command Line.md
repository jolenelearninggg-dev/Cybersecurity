## Gathering System Information
`ipconfig`：查看自己的網路配置，顯示機器目前的 TCP/IP 網路設定。

- ➔ 電腦的 「基本資料」

`arp -a`：查看這台電腦最近跟誰「說過話」，查看 ARP 快取，它會列出這台機器最近接觸過的其他主機 IP 地址與實體位址 (MAC)。

- ➔ 電腦的 「訪客紀錄」

`whoami` ： `ACADEMY-WIN11\htb-student` 顯示目前登入的 `[網域或電腦名稱(AD)] \ [使用者名稱]`。   

- ➔ 電腦的 「身分證」

`nltest /dclist:<網域>`：查詢網域控制站 (DC) 清單，雖然權限不足可能導致無法存取詳細資料，但仍能洩漏 DC 的主機名稱 (例如 `ACADEMY-ICL-DC`)。。     

- ➔ 查詢同網域內電腦的「控制者」

`whoami /priv`：顯示當前使用者的安全性特權（Privileges）與狀態（Enabled/Disabled）。

- ➔ 電腦的 「權限清單」
   |關注權限名稱|核心功能 (白話解釋)|為什麼駭客喜歡它？ (提權潛力)|
   |-----------|------------------|--------------------------|
   |`SeDebugPrivilege`|偵錯程式權限：允許你查看並存取系統中任何正在執行程式的記憶體。|最強權限之一。可以直接讀取 lsass.exe 進程來獲取系統中所有人的密碼雜湊值（Hashes）。|
   |`SeImpersonatePrivilege`|模擬權限：允許程式在驗證後「冒充」另一個客戶端的身分執行動作。|提權必看。可以配合「土豆（Potato）」系列漏洞，將自己的權限直接提升到最高階的 SYSTEM 帳號。|
   |`SeBackupPrivilege`|備份權限：允許你繞過檔案系統的所有限制，讀取硬碟上任何檔案。|無視權限限制。即使是加密或設定「只有老闆能看」的資料夾，你也能直接複製出來。|
   |`SeRestorePrivilege`|還原權限：允許你將檔案寫入或覆蓋到系統的任何位置，不受權限阻擋。|植入後門。可以覆蓋掉系統關鍵執行檔（如 sethc.exe），在不登入的情況下取得控制權。|
   |`SeTakeOwnershipPrivilege`|所有權取得權限：允許你強制奪取任何檔案或物件的「主導權」。|反客為主。你可以先把系統檔案的所有權改成你自己，再隨意修改它的內容或存取限制。|
   |`SeLoadDriverPrivilege`|加載驅動權限：允許你手動載入或卸載硬體驅動程式。|內核級攻擊。可以載入一個有漏洞的舊驅動，藉此進入作業系統的最底層（Kernel）執行指令。|


`whoami /groups`：查看當前使用者所屬的安全性群組。

 - ➔ 電腦的 「社會地位」

`net user`：顯示主機上所有的本地使用者帳號。
